## Accessing the API

Generate API keys in [Account Settings](https://www.traddocs.com/api-keys).


## Authentication

All API requests must include an `x-api-key` header with your API key.

```json cURL
curl https://api.traddocs.com/$TRADDOCS_ENDPOINT \
     -H "x-api-key: $TRADDOCS_API_KEY" \
     ...
```
